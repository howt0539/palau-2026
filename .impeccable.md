## Design Context

### Users
- 目前是 Howt 和 Ashley 使用的帛琉旅行伴侶 app
- 未來可能做成通用旅行 app 模板給其他人用
- 使用情境：旅行前規劃、旅行中記帳/記錄、旅行後回顧
- Mobile-first（iPhone Safari），透過 howt-base proxy 存取

### Brand Personality
**療癒、像素風、探險感**
- 像 Dave the Diver 的遊戲感，輕鬆可愛但不幼稚
- 深海主題配合像素藝術，營造沉浸式探險氛圍
- 工具本身要實用，但包裝要有溫度

### Emotional Goals
- **期待與興奮** — 旅行前的倒數感覺，每次打開都想到即將出發
- 不要冷冰冰、不要像企業軟體、不要表格感重的 UI

### Anti-references
- 企業軟體風格（冷灰色調、密集表格、無趣的 dashboard）
- 像素風可以，但不能太童趣/卡通化

### Aesthetic Direction
- **Theme:** 深海像素風 — 深藍漸層背景 + 上升泡泡 + 像素 icon
- **Color story:** 深海藍 (`#051b2e`) 為底，青色 (`#5ec4d4`) 為互動焦點
- **Surface:** Glassmorphism — 半透明毛玻璃卡片 `rgba(255,255,255,0.07)` + backdrop-blur
- **Icons:** 自製像素 SVG icon 系統（PX object），crispEdges rendering
- **Mode:** Dark only（深海主題不適合亮色模式）

### Design Principles

1. **Deep Sea Immersion** — 一切設計決策都要強化「潛入深海」的沉浸感。背景漸層、泡泡動畫、魚類圖鑑都是這個世界觀的一部分。
2. **Pixel Art Consistency** — 所有 icon 使用像素風格 SVG（shape-rendering: crispEdges + image-rendering: pixelated）。不混用 emoji 和像素 icon 在同一個語境中。
3. **Glass Over Solid** — 卡片和容器使用半透明毛玻璃效果，讓深海背景透出來。避免不透明的實色區塊打斷沉浸感。
4. **Tool First, Theme Second** — 記帳、行程記錄等核心功能的可用性優先。主題裝飾不能妨礙操作效率。輸入框、按鈕、列表的可讀性和點擊區域必須足夠。
5. **Mobile Thumb Zone** — 所有互動元素設計在拇指可及範圍。按鈕最小 44px 點擊區域。避免需要精確點擊的小目標。
